Assembly Language

Assembly Language or Assembly translates directly to machine language and is considered a low-level programming language.

Assembly is more human readable than machine code because it uses letters and numbers which are hexadecimal or decimal for operations rather than a number to define an operation in machine language.

Assembly doesn't have abstractions. Rather it's notation consists entirely of the instruction set that a given CPU provides. Some of the features lost between a high-level language and assembly are outlined below:

  • Assembly doesn't have variables, and most of the time doesn't have types.
  • Assembly doesn't support structs, classes, or objects
  • Assembly doesn't support methods or functions, rather they are made up of a large number of instructions.
  • An assembler is used to translate assembly language into machine language.

Assembly instructions use registers as inputs and outputs.

Store Word Instruction

The store word instruction copies a value from a register to memory. It can be written by writing sw then a register which acts as the source, a comma, then a constant and another register within parenthesis. The second register should hold a memory address where the contents of the first register will be stored. For example sw $t1, 0($t2). The constant plus the memory address is where the contents of the first register will be stored.

Control Flow Labels

Assembly uses labels as names for a location in a program which then allows manipulation of the control flow.

A control flow label can simply have a name followed by a colon. For example my_label:. To use a label, use the jump instruction.

Load Word Instruction

The load word instruction copies a value from memory to a register, and operates very similarly to the store word instruction with the difference that the first register location indicates the destination register for the value, and the second register holds the memory address to pull the value from. For example: lw $t3, 0($t4).

The load word instruction can be used to read from I/O devices, such as a switch.

Load Immediate Instruction

The load immediate instruction allows a value to be put into a register. The instruction is written by writing li then the register, then a comma and a value. For example li $t0, 8.

li takes two clock cycles to complete because it is a pseudo-operation made up of two separate native instructions.
